⌨ Controls
  • drag — pan the camera
  • wheel — zoom
  • click unit — select it (stacks show a unit list)
  • WASD / arrows / click neighbor — move or attack
  • hover an enemy — combat odds preview
  • hover with settlers — city site rating + footprint
  • click city — open the city view
  • double-click city — city view even with units on the tile
  • B — found city (settlers)
  • F — fortify unit
  • N — next idle unit (skips fortified)
  • 1 / 2 / 3 — quick unit production
  • C — cycle buildings / wonders
  • T — cycle research
  • click research bar — research panel
  • double-click — pick production / research instantly
  • F5 / F9 — quick save / quick load
  • Shift+S / Shift+L — save file / load file
  • drop a save file — load it
  • E / Enter — end turn
  • Esc — close panels
