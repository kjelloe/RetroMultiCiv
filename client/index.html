⌨ Controls
  • drag — pan the camera
  • wheel — zoom
  • click unit — select it
  • click neighbor tile — move / attack
  • click city — open the city view
  • B — found city (settlers)
  • F — fortify unit
  • N — next unit with moves
  • 1 / 2 / 3 — quick unit production
  • C — cycle buildings / wonders
  • T — cycle research
  • S / L — quick save / load
  • Shift+S / Shift+L — save file / load file
  • drop a save file — load it
  • E / Enter — end turn
  • Esc — close panels
